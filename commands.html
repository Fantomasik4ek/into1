Список встроенных команд

После того как ты их ввел в чат команды, вызывающие режим, жми Alt+P для начала

/патруль - эта команда запускает режим патруля

/упатруль - эта команда запускает режим для приёма учебного патруля

/рейд - эта команда запускает режим рейда

/пост - эта команда запускает режим поста города

/опрос - эта команда запускает меню с выбором опроса

/пейнтбол - эта команда запускает режим пейнтбола

/мг - проверка на наркотики

/багажник - проверка багажника

/алко - отыгровка алкотестера

/бсобес - покажет во сколько будет ближайшее собеседование

/ку - отыгровка представления без просьбы предоставить паспорт

/пропуск - отыгровка выдачи пропуска в ПД

/блокпост - запускает режим блокпоста

/on - включает автоматическую отыгровку оружия

/off - выключает автоматическую отыгровку оружия

/идея - команда чтобы сообщить разработчику о своей идее

/вопрос - напишите эту команду,чтобы задать вопрос поддержке, ответ ты увидишь после перезапуска биндера

Список мною добавленных команд

Помни, что их надо вводить после нажатия shift+f6

меню - тут ты найдешь много интерессного и сможешь запустить почти все режимы

пневмат - достать\убрать пневмат (помни про автоотыгровку оружия)

опрос - запускает меню с выбором опроса

отказ - используется, чтобы отказать на опросе

патруль - запускает режим патруля

рейд - запускает режим рейда

пост - запускает режим поста города

экзамены - открывает меню экзаменов

БП - запускает режим блокпоста

УП - эта команда запускает режим для приёма учебного патруля

ЗР - эта команда запускает режим для приёма зачета рядового

ЭК - эта команда запускает режим для приёма экзамена капиана

дверь - отыгровка выбрасывания человека из машины, перед выбрасыванием надо зафиксировать игрока (забиндите ее на какую-нибудь кнопку для удобства)

кхм - отыгровка закрытия МГ в IC чате (рекомендуется забиндить ее на какую-нибудь кнопку)

прощание - прощание с гражданином (тоже можно забиндить на какую-нибудь кнопку)

МП - команда поможет вам открыть нужный режим и доклад для проверки или повтора

тренировки - открывает меню тренировок

пейнтбол - запускает режим пейнтбола

полигон - запускает режим полигона

Дуэль - запускает режим Дуэли

заправка - доклады для заправки ТС

отсчет - отыгровка отсчета
