Информация о макросах

Макрос - Это заранее запрограммированое действие в
биндере, которое выполняеется при вызове макроса

Для использования макроса в отыгровке нужно написать
слово Macros далее в скобках его параметры
У разных макросов есть свои параметры и
разное их колличество, но первый параметра обязательно
название этого макроса, Macros(название,параметры)
Список всех макросов используемых в Tabirat:
Macros(Goto_Me,0) - Вести за собой зафиксированного игрока
Вторым параметром может быть все что угодно, на выполнение макроса
параметр не влияет

Macros(Delo,статья) - Заводит дело по статье указанной вторым параметром
на зафиксированного игрока

Macros(ShtrfAK,статья) - Выписывает штраф по статье АК указанной вторым параметром
зафиксированному игроку

Macros(ShtrfPDD,статья) - Выписывает штраф по статье ПДД указанной вторым параметром
зафиксированному игроку

Macros(Scrin,статья) - Делает скриншот, вторым параметром может быть что угодно

Macros(Dialog,текст) - Открывает диалоговое окно с указанным текстом

Macros(DialogEdit,текст:переменная) - Открывает диалоговое окно с указанным текстом
а так же добавляет поле ввода которое сохраняет значение в указанную переменную

Macros(SetValue,переменная:значение) - Присваивает выбранной переменной указанное значение

Macros(IfValue,переменная:значение:команда) - Проверяет если указанная переменная
равна указанному значению то выполняет выбранную комманду

Macros(IfNotValue,переменная:значение:команда) - Проверяет если указанная переменная
не равна указанному значению то выполняет указанную комманду

Macros(Random,переменная:минимум:максимум) - Заносит случайное значение в
указанную переменную в промежутке из минимального до максимального значения

Macros(Block,порядковый_номер) - Устанавливает дорожный блок
В параметре порядковый_номер указывается номер блока по счету

Macros(Timer,команда:время) - Задает таймер который будет выполнять
указанную команду через указанное время

Macros(TimerOff,0) - Выключает созданный таймер созданный
макросом Timer, вторым параметром может быть что угодно

Macros(ChekMenu,Название_пункта:задержка) - кликает на указанный пункт меню
Задержку можно не указывать, тогда она будет по умолчанию 50

Macros(PlayAudio,путь_к_файлу:wite) - Воспроизводит звуковой файл в формате wav
если убрать параметр wite то отыгровка будет продолжаться вместа с
воспроизведением файла, если использовать wite то скрипт будет
ждать окончания воспроизведения и только потом продолжит отыгровку

Macros(Сложить,переменная:число1:число2) - записывает в переменную сумму числа1 и числа2

Macros(Вычесть,переменная:число1:число2) - записывает в переменную значение вычитания из числа1 число2

Macros(Умножить,переменная:число1:число2) - записывает в переменную произведение числа1 на число2

Macros(Делить,переменная:число1:число2) - записывает в переменную результат деления числа1 на число2
